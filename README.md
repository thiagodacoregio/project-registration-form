# Stars of Tomorrow - School Registration Form

A modern and intuitive registration form for the Stars of Tomorrow early childhood education school. This project provides a seamless enrollment experience for parents registering their children.

## Features

- Clean and professional user interface
- Multi-step registration process
- File upload functionality
- Real-time form validation
- Responsive design
- Interactive form elements

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Poppins)
